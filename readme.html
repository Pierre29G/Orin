Ce dépôt contient un prototype pour la carte principal du projet. Il contient une installation de wordpress ainsi que les plugins nécessaire à la bonne éxecution du code. Une modification de wp-config, un emplacement pour une base de donnée SQL ainsi qu'un token pour l'API de map box à place dans le fichier \wp-content\themes\orin\assets\js\map.js sont nécessaire au bon fonctionnement du site.
